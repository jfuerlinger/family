# Tasks

**Tasks** is where your family organizes everything that needs doing — from the weekly groceries to planning the next vacation. Everyone in the family sees the same lists and tasks.

## Lists

Tasks live in lists. Create one list per topic, e.g. "Groceries", "Household", or "Vacation 2026".

### Create a list

1. Open **Tasks** in the menu.
2. Tap the button to create a new list.
3. Enter a name and pick a **color** — it makes the list easy to spot later.
4. Tap **Create**.

### Edit or delete a list

- **Edit:** Open the list and choose **Edit** to change its name or color.
- **Delete:** Choose **Delete**. Careful: this also removes every task in the list. You'll be asked first: "Really delete? This cannot be undone."

## Adding tasks

1. Open a list.
2. Tap **Add**.
3. Enter a title — that's all you need! Optionally you can also set:
   - **Due date** — when the task should be done
   - **Priority** — Low, Medium, or High
   - **Assignee** — the family member who'll take care of it
   - **Notes** — details, e.g. "the lactose-free milk!"
4. Tap **Save**.

> **Tip:** Tasks with a due date show up on the **Dashboard** — flagged as **Due today** or **Overdue** — so nothing slips through.

## Priorities

| Priority | When to use it |
| --- | --- |
| **High** | Needs to happen soon — e.g. "Gift for Grandma's birthday tomorrow" |
| **Medium** | Everyday stuff (the default) |
| **Low** | Someday — e.g. "Declutter the basement" |

## Assigning tasks

Assign a task to a family member so it's clear who's on it. Just pick the person when creating or editing the task. Each task can be assigned to exactly one person — or to nobody, in which case: whoever sees it, does it. 😉

## Completing tasks

Tap the checkbox in front of a task to mark it done. Tap again to undo — in case the dishwasher wasn't actually emptied after all.

## Editing or deleting tasks

- **Edit:** Open the task and choose **Edit**. Title, notes, due date, priority, and assignee can be changed at any time. Confirm with **Save**.
- **Delete:** Choose **Delete** if a task should disappear entirely.

> **Note:** Everyone in the family can see, complete, edit, and delete every task — FamilyHub runs on trust, not permissions.
