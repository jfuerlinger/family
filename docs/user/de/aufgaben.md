# Aufgaben

Unter **Aufgaben** organisiert ihr alles, was in der Familie zu erledigen ist – vom Wocheneinkauf bis zur Urlaubsplanung. Alle Familienmitglieder sehen dieselben Listen und Aufgaben.

## Listen

Aufgaben leben in Listen. Lege für jedes Thema eine eigene Liste an, z. B. „Einkaufen", „Haushalt" oder „Urlaub 2026".

### Liste erstellen

1. Öffne **Aufgaben** im Menü.
2. Tippe auf die Schaltfläche zum Anlegen einer neuen Liste.
3. Gib einen Namen ein und wähle eine **Farbe** – so erkennt ihr die Liste später auf einen Blick.
4. Tippe auf **Erstellen**.

### Liste bearbeiten oder löschen

- **Bearbeiten:** Öffne die Liste und wähle **Bearbeiten**, um Name oder Farbe zu ändern.
- **Löschen:** Wähle **Löschen**. Achtung: Damit verschwinden auch alle Aufgaben in der Liste. Du wirst vorher gefragt: „Wirklich löschen? Das kann nicht rückgängig gemacht werden."

## Aufgaben anlegen

1. Öffne eine Liste.
2. Tippe auf **Hinzufügen**.
3. Gib einen Titel ein – das reicht schon! Optional kannst du außerdem festlegen:
   - **Fällig am** – ein Datum, bis wann die Aufgabe erledigt sein soll
   - **Priorität** – Niedrig, Mittel oder Hoch
   - **Zugewiesen an** – ein Familienmitglied, das sich kümmert
   - **Notizen** – Details, z. B. „die laktosefreie Milch!"
4. Tippe auf **Speichern**.

> **Tipp:** Aufgaben mit Fälligkeitsdatum tauchen auf der **Übersicht** auf – inklusive der Hinweise **Heute fällig** und **Überfällig**, damit nichts untergeht.

## Prioritäten

| Priorität | Wann? |
| --- | --- |
| **Hoch** | Muss bald passieren – z. B. „Geschenk für Omas Geburtstag morgen" |
| **Mittel** | Der normale Alltag (Standard) |
| **Niedrig** | Irgendwann – z. B. „Keller ausmisten" |

## Aufgaben zuweisen

Weise eine Aufgabe einem Familienmitglied zu, damit klar ist, wer dran ist. Beim Anlegen oder Bearbeiten wählst du dazu einfach die Person aus. Jede Aufgabe kann genau einer Person zugewiesen sein – oder niemandem, dann gilt: Wer sie sieht, darf sie machen. 😉

## Aufgaben erledigen

Tippe auf das Kästchen vor einer Aufgabe, um sie als erledigt zu markieren. Nochmal tippen macht es rückgängig – falls die Spülmaschine doch nicht ausgeräumt war.

## Aufgaben bearbeiten oder löschen

- **Bearbeiten:** Öffne die Aufgabe und wähle **Bearbeiten**. Titel, Notizen, Fälligkeit, Priorität und Zuweisung lassen sich jederzeit ändern. Mit **Speichern** bestätigen.
- **Löschen:** Wähle **Löschen**, wenn eine Aufgabe ganz weg soll.

> **Hinweis:** Alle in der Familie können alle Aufgaben sehen, abhaken, bearbeiten und löschen – FamilyHub setzt auf Vertrauen statt Berechtigungen.
